Ocaml

A functional programming language in the ML family.

More info

No more crying at runtime

Ocaml is much better than everything else. Seriously, the programs that are written in functional languages like Ocaml are easy to understand and write. The type system is so expressive and powerful that you have a very good idea of how your program will run after it is compiled. I look forward to seeing where the Reason React project ends up. Proper types on the front end are definitely something we should care about.

TypeScript has brough sanity to web dev

The amount of times that JavaScript's type system has made me lose my mind cannot be overstated. While TypeScript cannot necessarily enforce types in the way that a language like Ocaml can, it serves its purpose in a world of computing which needs a little bit of flexibility. TypeScript might be one of the best things Microsoft has ever worked on. Sure beats Windows.
